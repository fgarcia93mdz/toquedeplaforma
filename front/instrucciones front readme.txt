Frontend de App de Terminal Omnibus

Montado en el framework/libreria React

Se creo usando yarn como package manager usando el comando 

"yarn create react-app"


INSTRUCCIONES:

Luego de clonar el repositorio desde github

1. Instalar dependencias con "yarn install"

2. Correr el proyecto con "yarn start"



(Use yarn porque es una version mejorada y mas rapida del antiguo npm install, npm run dev, etc)
